Resumo HTML – Radio

1. O que é um botão radio?

O botão do tipo radio permite ao usuário selecionar apenas uma opção entre várias disponíveis.

Ele é criado com a tag <input type="radio"> e todas as opções que pertencem ao mesmo grupo devem ter o mesmo atributo name.

2. Exemplo básico

<input type="radio" name="genero" value="feminino">
<input type="radio" name="genero" value="masculino">
<input type="radio" name="genero" value="outro">

3. Regras importantes

  • Todos os radio com o mesmo name formam um grupo
  • Apenas uma opção pode ser selecionada por grupo
  • O atributo checked pode ser usado para definir uma opção marcada por padrão
  • O atributo required obriga o usuário a escolher uma das opções

4. Diferença entre radio e checkbox

  • Radio: só uma opção pode ser escolhida
  • Checkbox: várias opções podem ser marcadas

6. Resumo das tags usadas

  • <input type="radio"> – cria o botão de seleção única
  • <label> – melhora a acessibilidade e permite clicar no texto
  • <fieldset> e <legend> – organizam grupos de opções
  • name – obrigatório para o agrupamento
  • value – valor enviado quando a opção é selecionada
  • checked – pré-seleciona uma opção
  • required – exige que uma das opções seja marcada
