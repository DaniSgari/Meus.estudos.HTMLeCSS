Resumo: Tipos de Mídia com Media Queries

O que são tipos de mídia?

Os tipos de mídia definem o tipo de dispositivo para o qual o CSS será aplicado. São usados com @media ou no atributo media das tags <link> ou <style>.

Tipos de mídia mais usados

Tipo Uso
screen Aplica estilos para telas (computadores, celulares, tablets)
print Estilos para impressão ou visualização do modo imprimir
all Aplica estilos para todos os dispositivos (padrão)
speech Leitores de tela ou softwares de voz

Exemplo usando @media

/* Estilo para telas */
@media screen {
  body {
    background-color: lightgray;
  }
}

/* Estilo para impressão */
@media print {
  body {
    background-color: white;
    color: black;
  }
}

/* Estilo para leitores de tela */
@media speech {
  h1 {
    speak: literal-punctuation;
  }
}

Exemplo com o atributo media no HTML

<link rel="stylesheet" href="style.css" media="screen">
<link rel="stylesheet" href="print.css" media="print">

Outros tipos de mídia (menos comuns)

  • braille – dispositivos táteis para leitura
  • embossed – impressoras braille
  • handheld – dispositivos portáteis antigos
  • projection – projetores (ex: slides)
  • tty – terminais de texto (sem imagem)
  • tv – navegadores de TV

⚠️ Importante: Esses tipos são pouco usados hoje em dia. Os mais relevantes são screen, print e all.
