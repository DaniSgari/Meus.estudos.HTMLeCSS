Resumo HTML – Checkboxes

1. O que é um checkbox?

Um checkbox (caixa de seleção) permite que o usuário selecione uma ou mais opções em um formulário.

Ele é criado com a tag <input type="checkbox">.

2. Exemplo básico

<input type="checkbox" name="opcao1"> Aceito os termos

3. Várias opções (grupo de checkboxes)

Você pode usar vários checkboxes com o mesmo name (como array) ou nomes diferentes, dependendo do envio. 

<input type="checkbox" name="interesses" value="html"> HTML
<input type="checkbox" name="interesses" value="css"> CSS
<input type="checkbox" name="interesses" value="js"> JavaScript

4. Atributos úteis em checkbox

  • name – nome do campo (usado no envio dos dados)
  • value – valor que será enviado se o checkbox for marcado
  • checked – define o checkbox como marcado por padrão
  • disabled – desativa o checkbox
  • required – exige que pelo menos um
