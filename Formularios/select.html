Resumo HTML – Elementos de Formulário além do <input>

1. <textarea> – Área de texto

Permite ao usuário digitar várias linhas de texto, como comentários ou descrições.

<textarea name="mensagem" rows="4"></textarea>

2. <select> e <option> – Lista suspensa

Permite ao usuário escolher uma opção de uma lista.

<select name="estado">
  <option value="sp">São Paulo</option>
</select>

3. <datalist> – Lista de sugestões

O <datalist> permite criar uma lista de sugestões automáticas para um campo input. Ele não restringe as opções, apenas sugere. 

<input list="cursos" name="curso">
<datalist id="cursos">
  <option value="HTML">
  <option value="CSS">
</datalist>

O usuário pode digitar livremente ou escolher uma opção da lista sugerida.
