Resumo HTML – Uso da tag <output> em Formulários

1. O que é a tag <output>?

A tag <output> é usada para exibir resultados calculados dentro de um formulário. Ela pode ser atualizada por JavaScript ou conectada a campos de entrada via atributos como for.

2. Exemplo com input range

50 
<form oninput="resultado.value=slider.value">
  <input type="range" id="slider" name="slider" value="50">
  <output name="resultado" for="slider">50</output>
</form>

3. Exemplo de soma de dois números

Resultado: 0

<form oninput="res.value = parseInt(a.value) + parseInt(b.value)">
  <input type="number" name="a">
  <input type="number" name="b">
  <output name="res" for="a b">0</output>
</form>

4. Explicação dos atributos

  • name: identifica o output no script
  • for: indica de quais inputs o output depende
  • oninput: executa uma ação toda vez que o formulário sofre mudança
